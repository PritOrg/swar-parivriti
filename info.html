How to Use the Note Transposition Tool

1. Setting Up Scales

Enter the Original Scale (e.g., C) and the Flute Scale (e.g., G) to set up the scale transposition. This defines the base scale to transpose notes from and the target scale to transpose notes to.

2. Entering Notes

Type your musical notes in the input area. Notes should be separated by commas or spaces (e.g., C, D, E).

If you enable the Rajvi Special switch, the input format changes. Use double periods ("..") to separate notes (e.g., P .. P .. D .. S'). Specific Gujarati notes can be input using Gujarati pronunciations.

3. Viewing Results

After pressing the Transpose Notes button, you'll see the transposed notes displayed in both Western and Gujarati formats. Copy the notes for further use!

4. Additional Features

When "Rajvi Special" is active, additional features like Gujarati-specific notation become available, allowing you to input notes with special symbols (e.g., S' for S#).
